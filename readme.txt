This plugin forces the site to load over https version of the website, using 301 redirect.

== Description ==
Using this plugin you can force the site to load over ssl. It redirects the user to the https version of the website.

== Installation ==


    Upload `Simple Force SSL` to the /wp-content/plugins/ directory

    Activate the plugin through the ‘Plugins’ menu in WordPress

    Edit any page and add destination URL to the option metabox on the top right.
	
	For removing a redirect, simply blank out the field.
